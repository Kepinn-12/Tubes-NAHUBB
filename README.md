# Tubes_NAHUBB

## General Information
Program dibuat menggunakan bahasa Java guna memenuhi "Tugas Besar  Stima 2025/2026". Algoritma greedy yang diimplementasikan ialah menggunakan pendekatan nilai heuristik untuk tiap aksi yang dapat dilakukan oleh bot pada , kemudian mengambil aksi dengan nilai heuristik tertinggi.

## Program Requirement
* **Java 17** <br>
* **Visual Studio  Code** <br>
* **Robocode** <br>
* **.Net 10** <br>

## Getting Started

1.  Clone Repository
   ```sh
   git clone https://github.com/Kepinn-12/Tubes-NAHUBB.git
   ```
2.  Buka Project
    - Gunakan VS Code
    - Pastikan sudah berada pada file tubes
    
3.  Build Project (Bot) 
    - Buka terminal.
    - Masuk ke dalam file bot pertama
    - Build dengan 'dotnet build'
    - Lakukan untuk semua bot

4. Buka cmd pada folder "Target", bot dapat dijalankan dengan:
   ```sh
   java -jar JavaBot.jar
    ```
5.  Install Bot ke Robocode
    - Buka menu config pada robocode.
    - Pilih file src

6.  Jalankan Robocode
    Buka aplikasi Robocode
    Pilih:
    - Battle 
    - Cari nama bot untuk diboot

7.  Run Battle
    - Tambahkan bot ke arena
    - Klik Start Battle
